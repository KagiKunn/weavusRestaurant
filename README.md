# weavusRestaurant
SpringBoot Project

## ⏱️Development Period
*2025-03-06 ~ 2025-03-12

## 🛠️Detail
Restaurant reservation system

### User
- Sign in
- Sign up
- Update profile
- Delete account
- Show restaurant information
- Restaurant reservation
- Cancel reservation
- Recommend Restaurant

### Owner
- All features of the user
- Registry Restaurant
- Update Restaurant
- Delete Restaurant
- Confirm Reservation
- Cancel Reservation
